| Token | Value | Usage |

| sm | 8px | Input inner padding (vertical), helper text gap |

## Color

Palette direction: greens + blues — trust, community, local feel (user's choice).

| Role | Hex | Usage |
|------|-----|-------|
| Dominant (60%) | `#FAFAFA` | Page background, auth page canvas |
| Secondary (30%) | `#F0FDFA` | Auth card surface, input backgrounds on focus |
| Accent (10%) | `#0D9488` | Primary CTA buttons, active states, focus rings |
| Destructive | `#DC2626` | Inline field validation errors, logout confirm |

**Accent reserved for exactly:**
1. Primary CTA button fill ("Entrar", "Criar conta")
2. Focus ring on all interactive elements (2px ring, 2px offset)
3. Hyperlinks within form area ("Esqueci minha senha", "Já tenho conta")
4. Checkbox checked state

**Never use accent on:** disabled states, placeholder text, decorative elements.

**Semantic palette mapping to shadcn CSS variables:**

```css
/* Mapped to shadcn new-york CSS variable names */
--background: #FAFAFA;          /* dominant */
--card: #F0FDFA;                /* secondary */
--primary: #0D9488;             /* accent */
--primary-foreground: #FFFFFF;
--destructive: #DC2626;
--destructive-foreground: #FFFFFF;
--border: #E2E8F0;
--input: #E2E8F0;
--ring: #0D9488;                /* focus ring = accent */
--foreground: #0F172A;          /* slate-900, body text */
--muted: #94A3B8;               /* placeholder text */
--muted-foreground: #64748B;
```

## States & Interactions

### Form Field States

| State | Visual Treatment |
|-------|----------------|
| Default | Border `#E2E8F0`, background white |
| Focus | Ring 2px `#0D9488`, border `#0D9488` |
| Error | Border `#DC2626`, `FormMessage` below field in `#DC2626` |
| Disabled | Opacity 50%, cursor not-allowed |
| Loading (submit) | CTA button shows spinner (Lucide `Loader2` + `animate-spin`), disabled |

### Button States

| State | Visual Treatment |
|-------|----------------|
| Default | Background `#0D9488`, text white, shimmer animation (Magic UI) |
| Hover | Background `#0F766E` (teal-700) |
| Active/Pressed | Background `#115E59` (teal-800) |
| Loading | Spinner replaces label text, disabled, opacity 80% |
| Disabled | Background `#94A3B8`, cursor not-allowed |

### Auth Flow States

| Scenario | UI Response |
|----------|------------|
| Login success | Redirect to `/` (home), no toast needed |
| Login failure (wrong credentials) | `Alert variant="destructive"` above form: "E-mail ou senha incorretos." |
| Login failure (network error) | `Alert variant="destructive"`: "Erro de conexão. Tente novamente." |
| Register success | Redirect to `/` (or profile setup prompt in Phase 2) |
| Register failure (email taken) | `Alert variant="destructive"`: "Este e-mail já está cadastrado." |
| Register failure (validation) | Inline `FormMessage` per field |
| Protected route, unauthenticated | Silent redirect to `/login` — no error message |
| Logout | Redirect to `/login` — no confirmation modal needed (non-destructive) |

## Accessibility Contract

| Requirement | Implementation |
|-------------|---------------|
| Focus visible | 2px ring `#0D9488`, 2px offset — never `outline: none` |
| Color contrast | `#0F172A` on `#FAFAFA` = 16.4:1 ✓ · `#FFFFFF` on `#0D9488` = 4.6:1 ✓ |
| Error association | `aria-describedby` linking `FormMessage` id to input |
| Loading state | `aria-busy="true"` on form during submission |
| Required fields | `aria-required="true"` on all auth inputs |
| Password toggle | (optional) Eye icon button with `aria-label="Mostrar senha"` |
| Keyboard nav | All interactive elements reachable and operable via Tab/Enter/Space |
